# Bank Customer Churn Prediction

## Overview

This project predicts whether a bank customer is likely to churn using Machine Learning algorithms.

The project was developed as part of an MBA project on Predictive Modeling and Risk Scoring.

---

## Technologies Used

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Google Colab
- Microsoft Excel

---

## Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

---

## Features

- Customer Churn Prediction
- Risk Classification
- Churn Probability
- Interactive Streamlit Web Application
- Excel Dashboard
- Machine Learning Model Comparison
